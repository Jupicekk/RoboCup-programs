# RoboCup programy

Zbierka robotických programov pre RoboCup.

## Obsah

- **Mindstorm programy/** — LEGO MINDSTORMS Robot Inventor (`.lms`)
  - `Bomb (NEW).lms`, `Bomb (NEW) - Python.lms`, `Bomb TEST 1.4.lms`
  - `Prasa Main (NEW).lms`, `Prasa Main (NEW) - Python.lms`
- **EV3 programy/** — LEGO EV3 (`.lmsp`)
- **MicroBit/** — micro:bit programy (`.hex`)
  - `CuteBot/`, `Srdce svetielka/`

## Roboty

- **Bomb** — 4-kolesový robot (predné B+A, zadné E+F), farebný senzor C. Komunikuje s Prasa cez BLE.
- **Prasa** — sleduje čiaru, ruka na motore E, reaguje na zelenú a vzdialenosť.
